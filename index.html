Coffee House ☕

Sistema básico de pedidos de Coffee House

Descripción

Coffee House ofrece una selección de cafés y productos de pastelería. A través de este sistema, el usuario puede armar su pedido eligiendo diferentes opciones del menú e incorporándolas a un carrito de compras.

Cada producto seleccionado se registra automáticamente, y al finalizar el proceso, el sistema muestra el total a abonar junto con el detalle completo de la compra.

¿Cómo funciona?

Al cargar la página, aparecerán ventanas emergentes donde podrás:

  • Ingresar el número del producto para agregarlo al carrito.
  • Escribir PAGAR para finalizar la compra.
  • Cancelar el pedido en caso de cerrar o dejar vacío el cuadro.
El sistema utiliza alert, prompt y confirm para la interacción, además de registrar datos en la consola de JavaScript.
